# python-expense-tracker
A beginner expense tracker project built using python

## Features
- Takes user input for:
  - Food expenses
  - Transport expenses
  - Entertainment expenses
  - Other expenses
- Calculates total daily spending
- Displays a summary of all expenses

## Concepts Used
- Variables
- User input
- Float data type
- Arithmetic operations
- Formatted strings (f-strings)

## Example Output

---- Daily Expense Tracker ----

Enter food expenses: $20
Enter transport expenses: $10
Enter entertainment expenses: $15
Enter other expenses: $5

Food expenses are: $20
Transport expenses are: $10
Entertainment expenses are: $15
Other expenses are: $5
Total expenses are: $50
